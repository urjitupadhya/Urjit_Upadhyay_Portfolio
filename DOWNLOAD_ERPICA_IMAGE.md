# Download Erpica Image Instructions

To add the Erpica landing page image to your local assets:

## Option 1: Manual Download
1. Right-click on the Erpica image you uploaded
2. Save it as: `erpica-landing-page.png`
3. Place it in: `assets/images/erpica-landing-page.png`

## Option 2: From GitHub (if you've uploaded it)
1. Go to your Erpica GitHub repository
2. Upload the image to: `assets/erpica-landing-page.png`
3. Update the image path in projects.user.json to the GitHub URL
